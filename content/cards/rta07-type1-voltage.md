> Voltage defect형 Type 1 RTA는 집합관 내강의 음전위 형성이 불충분해 수소 이온과 칼륨 이온이 함께 배설되지 못하므로 산증과 고칼륨혈증이 동시에 나타난다.

## ❶ 생각해 보셨나요?

Type 1 RTA라고 하면 distal nephron에서 산 배설 장애 → 저칼륨혈증이라고 외웠을 것이다. 그런데 Type 1 RTA면서 칼륨이 오히려 올라가는 아형(subtype)이 있다. 더 혼란스러운 것은 이 아형의 특징이 Type 4 RTA와 겹쳐 보인다는 점이다.

## ❷ 내강 음전위가 왜 산 배설과 칼륨 배설을 동시에 결정하는가?

집합관(collecting duct) 내강은 정상적으로 음전위(negative luminal potential)를 유지한다. 이 음전위는 주세포(principal cell)의 ENaC(epithelial sodium channel)을 통해 Na⁺가 내강에서 세포 안으로 들어오면서 형성된다. Na⁺가 양이온이므로 들어온 만큼 내강이 음성을 띤다.

이 음전위가 있어야 intercalated cell의 H⁺-ATPase와 ROMK 채널이 각각 수소 이온과 칼륨 이온을 내강으로 내보낼 수 있다. 양이온이 음전위 쪽으로 끌려가는 전기화학적 기울기 덕분이다.

Voltage defect형 Type 1 RTA에서는 ENaC 기능 이상으로 Na⁺가 내강에서 잘 흡수되지 않는다. Na⁺ 흡수가 줄면 내강 음전위가 감소한다. 전위가 작아지면 수소 이온도, 칼륨 이온도 내강으로 나가기 어려워진다.

1) 수소 이온 배설 감소 → 산 축적 → **산증(acidosis)**
2) 칼륨 배설 감소 → 혈청 칼륨 상승 → **고칼륨혈증(hyperkalemia)**

즉, 같은 distal nephron의 산 배설 장애지만, 원인이 H⁺-ATPase 자체의 문제가 아니라 전위 형성 실패라는 점이 다르다.

## ❸ 그렇다면 왜 Type 4가 아니라 Type 1로 분류하는가?

이 아형의 임상 양상은 산증 + 고칼륨혈증으로 Type 4 RTA와 겹쳐 보인다. 그러나 분류 기준은 임상 양상이 아니라 **주 장애 부위와 기전**이다.

- Type 1: distal nephron에서의 산 배설 장애가 기본 결함
- Type 4: 알도스테론 결핍 또는 저항성이 기본 결함

Voltage defect형은 알도스테론이 문제가 아니라 ENaC 채널 기능 이상으로 내강 전위가 낮아진 것이 핵심이다. 어쨌든 distal tubule에서의 산 배설 장애이므로 Type 1의 아형으로 분류한다. 그래서 **"voltage defect type Type 1 RTA"**라는 명칭이 붙은 것이다.

고칼륨혈증이 있다고 무조건 Type 4로 볼 수 없다. 기전을 확인해야 한다.

## ❹ 결국

Type 1 RTA의 전형은 H⁺-ATPase 결함 → 수소 이온 배설 장애 → 저칼륨혈증이다. Voltage defect형은 ENaC 이상 → 내강 음전위 소실 → 수소 이온과 칼륨 모두 배설 불가 → **고칼륨혈증을 동반한 산증**이다.

두 아형 모두 Type 1(distal 산 배설 장애)이지만 칼륨이 반대 방향으로 움직인다는 점에서, 칼륨으로 먼저 Type을 예단하면 오진으로 이어질 수 있다.
