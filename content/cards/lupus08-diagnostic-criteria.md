> 루푸스는 항핵항체(ANA) ≥1:80을 진입 기준으로 삼은 뒤, 다섯 개 도메인에 걸친 추가 기준 점수를 합산하여 10점 이상이면 진단한다.

## ❶ 생각해 보셨나요?

루푸스는 "전신 질환"이라고 하는데, 진단을 내리는 기준도 전신을 훑어야 할까? 피부에 나비 모양 발진만 있으면 진단할 수 있지 않을까?

실제로는 그렇지 않다. 특징적인 피부 발진 하나로 진단하기엔 비슷한 질환이 너무 많다. 루푸스 진단은 특정 소견 하나가 아니라, 여러 시스템에 걸친 이상 소견들을 점수로 누적해야 성립한다.

## ❷ 진단은 어떻게 구조화되어 있는가

2019년 ACR/EULAR 공동 가이드라인은 진단을 두 단계로 나눈다.

**1단계: 진입 기준 (Entry Criteria)**

ANA(항핵항체) ≥1:80이 먼저 충족되어야 한다. 이 검사가 음성이라면, 반복 검사에서도 음성이라면, 루푸스일 가능성은 매우 낮다고 판단한다.

**2단계: 추가 기준 (Additive Criteria)**

진입 기준을 통과한 뒤에는 여러 도메인에서 점수를 합산한다. **합산 점수 ≥10점이면 루푸스로 진단**한다.

도메인은 다섯 영역이다: 전신/혈액/신경정신, 피부·점막, 장막, 관절, 신장, 면역.

**도메인 내 점수 규칙**: 같은 도메인에서 항목이 여러 개 해당되더라도 가장 높은 점수 하나만 적용한다. 예컨대 백혈구 감소(3점)와 혈소판 감소(4점)가 동시에 있으면 7점이 아니라 4점이다.

## ❸ 각 도메인에서 점수는 어떻게 배분되는가

| 도메인 | 항목 | 점수 |
|--------|------|------|
| 전신/혈액 | 발열 | 2점 |
| | 백혈구 감소 (<4,000/μL) | 3점 |
| | 혈소판 감소 (<100,000/μL) | 4점 |
| | 자가면역성 용혈 | 4점 |
| 신경정신 | 섬망 | 2점 |
| | 경련 | 5점 |
| 피부·점막 | 급성 피부 루푸스 (malar rash) | **6점** |
| | 구강 궤양 | 2점 |
| 관절 | 활막염 동반 관절염 (2개 이상) | **6점** |
| 신장 | 단백뇨 ≥500mg/일 | 4점 |
| | 루푸스 신염 class III/IV | **10점** |
| 면역 | 보체 감소 (C3+C4 둘 다) | 4점 |
| | 보체 감소 (C3 또는 C4 하나) | 3점 |
| | 특이 항체 (anti-dsDNA, anti-Sm) | **6점** |

신장 class III/IV가 10점인 이유는, 루푸스 신염이 그만큼 중증이고 빠른 치료가 필수적이기 때문이다. 이 항목 하나만으로 진단 기준이 충족되는 것은 논란이 있으나, 그만큼 임상적 중요도가 높다는 반영이다.

## ❹ 결국 이 체계가 말하는 것

ANA 음성이면 진입 자체가 막힌다. 진입 후에는 단일 소견이 아니라 여러 시스템에 걸친 이상이 누적되어야 10점에 이른다.

루푸스가 "전신 질환"이라는 정의가 진단 체계에 그대로 반영되어 있다. 발열, 백혈구 감소, 경련, 피부 발진, 구강 궤양, 관절염, 신장 침범, 보체 감소 — 어느 한 부위가 아니라 여러 방향에서 동시에 이상이 쌓일 때 진단이 성립한다.
